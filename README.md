# Spring Cloud

### Technologies

*  Spring Boot 2.1.2.RELEASE
*  Spring Framework
*  Spring Data
*  Spring Cloud GCP
*  GCP AppEngine
*  GCP Cloud SQL

Source: https://github.com/spring-cloud/spring-cloud-gcp

### Running the application

`$ mvn spring-boot:run`
 
### Deploy to App Engine

 `$ mvn appengine:deploy` 

### Browse

 `$ gcloud app browse` 
 `$ gcloud app logs read | less`
